What I Do

Saya lulusan SMA dari kota kecil di Lampung dan bangga bisa meraih berbagai prestasi. Selama sekolah, saya berhasil jadi lulusan terbaik, bukan cuma di akademik tapi juga di kegiatan non-akademik.

Dulu, saya sering banget dijuluki "Babu Sekolah," dan jujur aja, yaaa nggak salah juga sih. Karena saya sendiri menilai kalo saya "budak korporat" sekolah yang tiap hari sibuk ngurusin ini-itu, dari A sampai Z. Tapi, dari situ juga saya belajar banyak hal dan jadi makin berkembang.

Sekarang saya kuliah di Institut Teknologi PLN dengan jurusan Teknik Informatika. Alhamdulillah, berkat semua pengalaman waktu jadi "budak korporat sekolah", saya berhasil mendapatkan program Ikatan Kerja PT. PLN (Persero). Jujur, lumayan nggak nyangka semua kerja keras dan kesibukan ngurusin berbagai hal dulu bisa ngebawa saya sampe sini.

Sedikit hal tentang website ini, rencananya semua kegiatan, pengalaman, tulisan, bakal saya kemas disini kalo gak males.

Saya suka dengerin lagu Surat Cinta Untuk Starla akhir-akhir ini, ntah kenapa. Saya juga suka dengerin Lany, Niki, Wave To Earth, Pasming Based JJ Select, dan SIMFONI GADJAH MADA, hehe.
